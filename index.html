Comandos Git — Guia Rápido

Resumo dos comandos Git mais usados com exemplos práticos.

git init
git init
Cria um novo repositório Git no diretório atual.
Exemplo

git init # cria .git e inicia controle de versão

git clone
git clone <url>
Baixa um repositório remoto para seu computador.
Exemplo

git clone https://github.com/usuario/repositorio.git

git status
git status
Mostra arquivos modificados, staged e não rastreados.
Exemplo

git status -s # status curto

git add
git add <arquivo> | . | -A
Adiciona mudanças à área de stage (prontas para commit).
Exemplo

git add . # adiciona todas as alterações

git commit
git commit -m "mensagem"
Salva as mudanças staged no histórico do repositório.
Exemplo

git commit -m "Corrige bug no login"

git log
git log --oneline --graph
Exibe o histórico de commits.
Exemplo

git log --oneline --graph --decorate --all

git branch
git branch <nome>
Gerencia ramos (branches). Para criar: git branch nome.
Exemplo

git branch feature/login

git checkout
git checkout <branch> | -b
Muda para outro branch. -b cria e alterna para o branch.
Exemplo

git checkout -b feature/x

git merge
git merge <branch>
Mescla mudanças de outro branch no branch atual.
Exemplo

git checkout main; git merge feature/login

git push / git pull
git push origin <branch>
Push envia commits ao remoto. Pull traz mudanças do remoto ao local.
Exemplo

git pull origin main

git reset
git reset --soft/hard <commit>
Move HEAD para outro commit; pode descartar mudanças (hard) ou preservar (soft).
Exemplo

git reset --soft HEAD~1

git stash
git stash save "mensagem"
Guarda alterações não commitadas temporariamente para mudar de branch com segurança.
Exemplo

git stash; git checkout main; git stash pop

git rebase
git rebase <branch>
Aplica commits de um branch sobre outro, reescrevendo o histórico.
Exemplo

git checkout feature; git rebase main

git remote
git remote add origin <url>
Gerencia repositórios remotos (adicionar, listar, remover).
Exemplo

git remote -v

Dica rápida: use git --help ou git --help para ver a documentação completa.
